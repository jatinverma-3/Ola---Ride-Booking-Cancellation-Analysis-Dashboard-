# Ola---Ride-Booking-Cancellation-Analysis-Dashboard-
This project focuses on analyzing ride booking data to understand booking patterns, ride cancellations, and overall ride performance. The dataset was cleaned and analyzed using SQL and Excel, and an interactive dashboard was developed in Power BI to visualize key business metrics.

Key Insights

Analyzed billing amount across different departments.
Evaluated treatment costs, medication costs, and insurance coverage.
Compared diagnosis-wise service distribution.
Identified revenue contribution from different hospital departments.

Dashboard Features

KPI cards for billing metrics
Department-wise billing analysis
Diagnosis and service type analysis
Geographic billing insights

Tools & Technologies

Power BI
Data Visualization
KPI Analysis
